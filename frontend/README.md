# D&B - Backend

### Requirements

- Node v12 LTS

### Installation

```
git clone https://github.com/Lemii/dungeons-and-beddows
cd ./dungeons-and-beddows/frontend
npm i
```

### Usage

#### Run in development mode

```
npm start
```

#### Run in production mode

```
npm i -g serve
npm run build
cd build
serve -s
```
